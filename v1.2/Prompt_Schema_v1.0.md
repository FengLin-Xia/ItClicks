# 🧠 Seed Engine Generation Control Schema v1.0

> **这是生成阶段唯一合法控制协议。以后不再随意加维度。**

---

## 0️⃣ 设计原则

- **五维控制**：各维独立
- **关系层 ≠ 表达层**
- **张力不是钩子**
- **RAG 只服务气质与结构**

---

## 1️⃣ 关系逻辑层（Relation Engine）

**控制“爱怎么成立”。**


| 字段          | 类型           |
| ----------- | ------------ |
| `primary`   | 张力类型         |
| `secondary` | 张力类型或 `null` |


**合法值（10 类）：**

- 不能爱
- 矛盾拉扯
- 反差（⚠️ 仅当作为关系矛盾存在时）
- 位阶张力
- 占有欲/吃醋
- 遗弃/等待
- 时间终止
- 伤害纠缠
- 行为悖论
- 情感留白

---

## 2️⃣ 表达形式层（Form）

**控制“文本怎么呈现”。**

```json
{
  "form": "single | dialogue | micro_scene | lyrical_blank"
}
```


| 值               | 说明       |
| --------------- | -------- |
| `single`        | 单句钩子     |
| `dialogue`      | 双人对话     |
| `micro_scene`   | 2–3 句微剧情 |
| `lyrical_blank` | 抒情留白     |


---

## 3️⃣ 表达强度层（Intensity）

**控制“情绪浓度”。**

```json
{
  "intensity": "low | medium | high"
}
```


| 值        | 说明   |
| -------- | ---- |
| `low`    | 克制暗示 |
| `medium` | 明显情绪 |
| `high`   | 极端表达 |


---

## 4️⃣ 钩子机制层（Hook Mechanism）

**控制“传播触发器”。**

```json
{
  "hook": "contrast | time_cut | behavioral_paradox | direct_confession"
}
```


| 值                    | 说明       |
| -------------------- | -------- |
| `contrast`           | 预期违背     |
| `time_cut`           | 时间终止/错过  |
| `behavioral_paradox` | 行为冲突     |
| `direct_confession`  | 直白告白（慎用） |


---

## 5️⃣ 风格气质层（Tone / Vibe）

**控制“语料对齐”。**

```json
{
  "tone": {
    "base_style": "east_asian_subtle",
    "coldness_level": 0,
    "restraint_level": 0,
    "concreteness_level": 0
  }
}
```

**推荐默认：**

```json
{
  "base_style": "east_asian_subtle",
  "coldness_level": 0.6,
  "restraint_level": 0.8,
  "concreteness_level": 0.5
}
```

> 数值范围：0–1

---

## 🔧 完整示例

```json
{
  "relation": {
    "primary": "位阶张力",
    "secondary": "情感留白"
  },
  "form": "dialogue",
  "intensity": "medium",
  "hook": "contrast",
  "tone": {
    "base_style": "east_asian_subtle",
    "coldness_level": 0.6,
    "restraint_level": 0.8,
    "concreteness_level": 0.5
  }
}
```

## 🎯 生成流程（必须遵守）

### Step 1：Plan 阶段（不可见）

模型先生成：

- 关系结构草图
- 钩子句位置
- 情绪峰值位置

### Step 2：Write 阶段

- 严格按参数写
- 不允许偏离 `tone`
- 不允许增加未授权张力

---

## 🧠 RAG 使用规则（写进协议）

**只允许对：**

- `tone.base_style`
- `form` 结构模板

做风格锚点提取。

**禁止：**

- 复制句子
- 拼贴内容
- 注入具体情节

---

## 🚫 禁止事项

- 不得新增第六维
- 不得混淆 `hook` 与 `relation`
- 不得同时使用 `high` intensity + `lyrical_blank`（冲突）
- 不得连续两次使用同一 `hook`（避免风格坍塌）

# 附录：张力类型详解

## 1️⃣ 不能爱（结构性阻碍）


| 项目       | 内容                                          |
| -------- | ------------------------------------------- |
| **定义**   | 双方情感真实存在；存在客观结构阻碍（身份/关系/立场/伦理）；爱成立，但不能实现或承认 |
| **核心感受** | 压抑、克制、不能公开                                  |
| **代表结构** | “不是不爱，是不能。” / “我爱你。” / “你不能爱我。”             |
| **生成提示** | 避免写成单恋或简单“爱而不得”。                            |


---

## 2️⃣ 矛盾拉扯（爱=痛）


| 项目       | 内容                     |
| -------- | ---------------------- |
| **定义**   | 靠近=毁灭；离开=死亡；情感与自保冲突    |
| **核心感受** | 撕裂、依附、共生、极端            |
| **代表结构** | “一靠近你我就痛。” / “离开你我会死。” |
| **生成提示** | 必须有冲突方向，不是纯悲情。         |


---

## 3️⃣ 反差（角色/情绪反转）


| 项目       | 内容               |
| -------- | ---------------- |
| **定义**   | 外在与内在反差；强者失控；冷→软 |
| **核心感受** | 破防、微妙、反转         |
| **代表结构** | 冷脸→撒娇 / 淡定→失态    |
| **生成提示** | 压缩表达，不解释背景。      |


---

## 4️⃣ 位阶张力（权力/年龄/左右位）


| 项目       | 内容                        |
| -------- | ------------------------- |
| **定义**   | 关系结构不对称；表达方式受位阶影响；欲望方向不平衡 |
| **核心感受** | 控制、训诫、示弱、摇尾               |
| **代表结构** | 年上训诫 / 年下求爱 / 左右位暗示       |
| **生成提示** | 位阶必须结合情感，否则只是设定。          |


---

## 5️⃣ 占有欲 / 吃醋


| 项目       | 内容             |
| -------- | -------------- |
| **定义**   | 排他性情感；微小行为触发不安 |
| **核心感受** | 酸、欲望泄露、情绪失控    |
| **代表结构** | “你看谁？” / 烟飘向别人 |
| **生成提示** | 避免写成暴力控制。      |


---

## 6️⃣ 遗弃 / 等待


| 项目       | 内容             |
| -------- | -------------- |
| **定义**   | 单向忠诚；被抛弃仍不离开   |
| **核心感受** | 执念、永恒、极端忠诚     |
| **代表结构** | “最后一次见你的地方等一生” |
| **生成提示** | 控制煽情强度。        |


---

## 7️⃣ 时间终止（临终 / 错过）


| 项目       | 内容                      |
| -------- | ----------------------- |
| **定义**   | 爱在终点成立；时间强化情绪           |
| **核心感受** | 晚了、不可逆、终局               |
| **代表结构** | 临终告白 / “我们喜欢上对方的时间恰好错过” |
| **生成提示** | 避免展开完整剧情。               |


---

## 8️⃣ 伤害纠缠（互相折磨）


| 项目       | 内容               |
| -------- | ---------------- |
| **定义**   | 伤害存在；仍然不分开；爱与恨混合 |
| **核心感受** | 残酷、病态浪漫、黏连       |
| **代表结构** | “互相折磨一辈子”        |
| **生成提示** | 不要走向纯黑暗暴力。       |


---

## 9️⃣ 行为悖论（敌意与温柔并存）


| 项目       | 内容                      |
| -------- | ----------------------- |
| **定义**   | 行为层面冲突；敌意与照料共存；情绪无法单一归类 |
| **核心感受** | 危险温柔、混乱浪漫               |
| **代表结构** | “杀你路上给你买早餐” / “恨你却替你挡刀” |
| **生成提示** | 必须有行为层冲突。               |


---

## 🔟 情感留白（日常酸涩）


| 项目       | 内容                             |
| -------- | ------------------------------ |
| **定义**   | 没有戏剧冲突；情绪克制；日常行为承载情感；爱没有被明确宣告  |
| **核心感受** | 淡、隐、余温、未说出口                    |
| **代表结构** | “好好吃饭。” / “注意身体。” / “像开在身边的花。” |
| **生成提示** | 避免写成平淡叙述，必须有隐含牵连。              |
